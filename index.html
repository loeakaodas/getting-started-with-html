Basics of the Web

The World Wide Web is a collection of HTML documents that are linked together by URLs and HTTP across the internet. The Web itself was invented in the early 90's at CERN and now comprises over 30 billion pages. You access the web through a browser, which sends a request across the internet using HTTP to a server; a specialiced computer designed to store websites. The most popular browsers are Internet Explorer, Chrome, Firefox.

HTML

HTML stands for Hypertext markup language. An HTML document would contain all the text, markup, refferences, and links to other pages that a browser would intepret to present a complete webpage to an end user.

Markup

HTML markup consits of tags & elements. A tag is something that can change the behavior of it's contents, for example making it bold or italic. It is important to remember to close all your tags or risk creating problems, there are however void tags that don't need to be closed: ex. the img or br tags. There are many different tags that do different things, you can for example link to the Udacity homepage! And many other fun things as well. A complete tag comprises an element.
A good thing to also remember is how whitespace is treated. All tabs, new lines, or multiple spaces get transtaled into a single space & that is why you need markup to change things up.

Computers are Stupid

Computers are very literal and logical (usually). They will take all your instructions exactly as they were written, no how they might have been intended. So it's important to follow the correct syntax when coding.

You can't remember everything!

So don't even try. Knowing how to find the right answer with a google search or documentation is always a wiser approach anyway.

Inline vs Block Elements

HTML treats certain elements as inline and other as block. What that means is that inline elements will apear on the same line, while

block elements get their own special "box" or block.
